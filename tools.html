Quartzified Tools

Quartzified Tools is a collection of Unity Editor Tools that I develope during the development of Aukedora to make development easier and/or faster. Each tool has its own seperate uses, providing each tool with some aspect to make game development easier.

All these tools are released for free on my Github page, and can easily be added to your Unity project by just downloading the latest release and importing the Unity Package into your project.

Quartzified Utility Library

The Quartzified Utility Library (QUL) || [COOL] is a Open source library of various useful methods/ functions that can be used in Unity and some DLLs even for general use.

QUL is completely free to be used and uses the GPL-3.0 license to ensure its open-source / free state.
